## Descrição

Projeto criado no NLW Journey da Rocketseat. 
O projeto consiste em um roteiro de viagens, com marcações de data, hora e se foi feito ou não.

- Biblioteca de Terceiros (Dayjs);
- Funções;
- Objetos e Estruturas de Dados;
- Eventos;
- Manipulação do DOM;
- Template Strings;
- Responsividade;
